--- title: Libraries sidebar: nav: - header: Official Libraries - text: Python href: #python - text: Ruby href: #ruby - text: PHP href: #php - header: Community Libraries - text: Java href: #community-java - text: Node.js href: #community-node - text: Perl href: #community-perl --- There are many libraries available to help you interact with the Datadog API.

Python

  • DogApi - A Python Datadog API wrapper.
  • dogstatsd-python - A Python Datadog Statsd client.

Ruby

  • DogApi - A Ruby Datadog API wrapper.
  • dogstatsd-ruby - A Ruby Datadog Statsd client.

PHP

  • php-datadogstatsd - A PHP Statsd client.

Community Libraries

Some great folks have written their own libraries to help interact with Datadog. Check them out:
  • Java
    • metrics-datadog - a backend to codahale's metrics library written by Vistar Media.
  • Node.js
    • node-dogstatsd is a Node.js client DogStatsd client, contributed by Young Han Lee.
  • Perl
    • dogstatsd-perl is a Perl DogStatsd client, contributed by Stefan Goethals.

If you've written a Datadog library, write us at code@datadoghq.com and we'll be happy to add it to the list.
