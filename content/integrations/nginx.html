Overview

Connect NGINX to Datadog in order to:
  • Visualize your web server performance
  • Correlate the performance of Nginx with the rest of your applications
From the open-source Agent: * NGINX YAML Example * NGINX checks.d The following metrics are collected by default with the NGINX integration: nginx.net.connections nginx.net.reading nginx.net.request_per_s nginx.net.waiting nginx.net.writing If you are using NGINX Plus, you can collect many more metrics, as described on the NGINX docs.
