公式ライブラリー

Datadog APIを操作するために、Datadogが公式に公開しているライブラリーです。

Python

  • DogApi - A Python Datadog API wrapper.
  • dogstatsd-python - A Python DogStatsD client.

Ruby

  • DogApi - A Ruby Datadog API wrapper.
  • dogstatsd-ruby - A Ruby DogStatsD client.

PHP

  • php-datadogstatsd - A PHP DogStatsD client.

C#

  • dogstatsd-csharp-client - A C# DogStatsD client.

コミュニティーライブラリー

Datadog APIを操作するために、コミュニティーのメンバーが開発/公開しているライブラリーです。

Java

  • java-dogstatsd-client - a DogStatsD Client for Java written by Indeed.
  • metrics-datadog - a backend to codahale's metrics library written by Coursera.
  • Lassie - a Java screenboard API client by Bazaarvoice.

Node.js

  • node-datadog - a Node.js API client, contributed by HashGo.
  • node-dogstatsd - a Node.js DogStatsD client, contributed by Young Han Lee.
  • node-dogapi - a Node.js API client, contributed by Brett Langdon.

Perl

  • webservice-datadog - a Perl API client, contributed by Jennifer Pinkham.
  • dogstatsd-perl - a Perl DogStatsD client, contributed by Stefan Goethals.

Ruby

  • metricks-dogstatsd - a backend for the popular Metriks gem, written by Mavenlink.

PHP

  • plesk_metrics_datadog - a PHP script to collect metrics from Plesk by Israel Viana.

Go

  • go-datadog-api is a Go wrapper for our API by Mark Smith.
  • go-dogstatsd is a dogstatsd client written in Go by Ooyala.

Python

  • scales_datadog is a Datadog backend for the Scales library written by Tommaso Barbugli.

インテグレーションライブラリー

Saltstack

  • Datadog Salstack Formula
  • Datadog Saltstack written by Luca Cipriani.

Ansible

  • This Ansible script is a callback plugin that posts events to your Datadog event stream as you deploy and is written by Alek Storm.

FreeSwitch

  • This is for a FreeSwitch ESL application to export statistics to DataDog using the dogstatsd API and is written by WiMacTel.

Google Analytics

  • You can get data into Datadog from Google Analytics using our API with this library.

Pid-stats

  • This library will allow you to generate process information from StatsD, given pid files. It was created by GitterHQ.

Logstash

  • This script will allow you to get data in from Logstash and was written by Datadog's very own Conor Branagan.

Reverse Color Scheme

  • To get a darker background than default, you can use this CSS library to flip the colors.

Datadogのライブラリーを開発した場合は、code@datadoghq.comへ是非ご連絡ください。 上記リストへ追加していきたいと思います。
