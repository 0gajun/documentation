--- title: Search --- Search for entities in Datadog by string. Currently, the searchable entities are `hosts` and `metrics`. ### Authentication Full access. (see [Authentication](/authentication/)) ### Parameters * `q`: the search query string (see Query Language below) ## Find all entities matching a string ### Request <%= request 'GET', '/api/v1/search?q=cassandra' %> ### Response <%= response_headers 200, 'Content-Type' => 'application/json' %> <%= http_body < ## Find hosts matching a string Use faceting to filter the request by type (see Query Language below for more details). <%= request 'GET', '/api/v1/search?q=hosts:cassandra' %> <%= response_headers 200, 'Content-Type' => 'application/json' %> <%= http_body < ## Find metrics matching a string Use faceting to filter the request by type (see Query Language below for more details). <%= request 'GET', '/api/v1/search?q=metrics:cassandra' %> <%= response_headers 200, 'Content-Type' => 'application/json' %> <%= http_body < ## Query language Search queries allow for limited faceting. Available facets are: * `hosts` * `metrics` Faceting your search limits your results to only matches of the specified type. Un-faceted queries return results for all possible types. Un-faceted queries are of the form: `query_string` Faceted queries are of the form: `facet:query_string`
