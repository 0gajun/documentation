Overview

Connect PostgreSQL to Datadog in order to:
  • Visualize your database performance.
  • Correlate the performance of PostgreSQL with the rest of your applications.
From the open-source Agent: * PostgreSQL YAML example * PostgreSQL checks.d The following metrics are collected by default with the PostgreSQL integration: postgresql.buffer_hit postgresql.commits postgresql.connections postgresql.disk_read postgresql.rollbacks postgresql.rows_returned postgresql.rows_fetched postgresql.rows_inserted postgresql.rows_updated postgresql.rows_deleted
