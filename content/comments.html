--- title: Comments ---

Post, edit, delete, and reply to comments.

Authentication

Full access. (see Authentication)

Post a new comment

Request

<%= request "POST", "/api/v1/comments" %> <%= request_headers 'Content-Type' => 'application/json' %> <%= http_body <

The handle parameter is optional. If unset the comment will be posted as the owner of the given application key.

Response

<%= response_headers 201, 'Content-Type' => 'application/json' %> <%= http_body <

Reply to a comment

Request

<%= request "POST", "/api/v1/comments" %> <%= request_headers 'Content-Type' => 'application/json' %> <%= http_body <

The handle parameter is optional. If unset the comment will be posted as the owner of the given application key.

Response

<%= response_headers 201, 'Content-Type' => 'application/json' %> <%= http_body <

Edit a comment

Request

<%= request "PUT", "/api/v1/comments/:comment_id" %> <%= request_headers 'Content-Type' => 'application/json' %> <%= http_body <

All fields are optional. Omitted fields will be unchanged.

Response

<%= response_headers 200, 'Content-Type' => 'application/json' %> <%= http_body <

Request

<%= request "DELETE", "/api/v1/comments/:comment_id" %>

Response
