--- title: Client Libraries --- ## Python The Python `dogapi` is a client library that wraps Datadog's raw HTTP API. It is designed to make it easy to report custom metrics from existing Python applications and build new tools on top of the API. * Download or install from PyPi: [http://pypi.python.org/pypi/dogapi](http://pypi.python.org/pypi/dogapi) * Documentation at ReadTheDocs: [http://dogapi.readthedocs.org/](http://dogapi.readthedocs.org/en/latest/index.html) * Source at GitHub: [https://github.com/DataDog/dogapi](https://github.com/DataDog/dogapi) ## Ruby The Ruby `dogapi` is a client library that wraps Datadog's raw HTTP API. It is designed to make it easy to report custom metrics from existing Ruby applications and build new tools on top of the API. * Download or install from RubyGems: [https://rubygems.org/gems/dogapi](https://rubygems.org/gems/dogapi) * Documentation at RubyDoc: [http://rubydoc.info/gems/dogapi/frames/](http://rubydoc.info/gems/dogapi/frames/) * Source at GitHub: [https://github.com/DataDog/dogapi-rb](https://github.com/DataDog/dogapi-rb) ## CLI The Python `dogapi` also provides a command line `dog` command that can be used to explore and interact with the Datadog API right from the terminal. The `dog` command is also ideal for scripting. It can provide data in a line-oriented format perfect for grepping and awking to your heart's content, and it works with shell redirection allowing you to easily send `/dev/random` directly to your event stream! * Download or install from PyPi: [http://pypi.python.org/pypi/dogapi](http://pypi.python.org/pypi/dogapi) * Documentation at ReadTheDocs: [http://dogapi.readthedocs.org/](http://dogapi.readthedocs.org/en/latest/index.html) * Source at GitHub: [https://github.com/DataDog/dogapi](https://github.com/DataDog/dogapi)
